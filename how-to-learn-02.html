学习《如何学习》—— 拖延症、记忆与睡眠

Parasomnia · · ·

了解拖延

这里只是初步了解，后面会再提到拖延的事情

拖延的原因

当你看到一件你极度不情愿做的事情时
就好像是你激活了大脑中与疼痛相关的区域
你的大脑就会自然而然地去寻找停止这种负面刺激的方式
而这一方式便是将你的注意力转移到其他事情上

拖延的过程

  1. 留意到可能会让你产生轻微的不安的某件事
  2. 你不喜欢这种不适
  3. 为了消除这种感觉会让自己不再关注手头上的这件事
  4. 转而去做一些更让人愉悦的事情 即拖延
  5. 这么做的结果就是短期内你觉得更开心了 即拖延成习惯

意志力工具 —— 番茄工作法

使用番茄工作法，你需要做到：

  1. 设定计时器为25分钟
  2. 屏蔽所有干扰
  3. 集中注意力
  4. 完成目标后给予自己奖励

记忆

抽象与具体

抽象：比如数学中的运算符号∫，在现实生活中不能直接找到它（尽管生活中有许多它的应用场景）
具体：比如英文单词meow，通过发音，你就能自然联想到猫的叫声（你见过猫，也听过猫叫）

但是如love，hope这一类的单词也在生活中找不到实例，也是一种抽象的概念，但为什么比数学符号要好理解得多呢？ —— 因为它们和我们的感情相关（即便我们看不到，也能感觉到）

实践使记忆永存

这意味着 练习在数学和科学中学到的思想和概念是很重要的
就像你学习任何其他东西一样 以此帮助你提高和增强你在学习过程中生成的神经连接
通过反复使用 神经元形成相互的连接 越是抽象的东西 越要注重练习
通过练习 这些概念才能为你所用 即使你涉及的概念是抽象的 但你创造的神经思维模式是真实且具体的

简而言之：刻意练习，熟能生巧

学习中的休息能使你事半功倍

你学习的时候想做的事情就是学习一些东西 通过一心一意地专注 拼命学习它
然后休息一下 或者至少稍稍将你的注意力转移到其他东西上
在这段表面上是放松的时间里 你大脑的发散模式 则有机会在后台工作 帮你完成对概念的理解

看起来似乎可以完美的和 番茄工作法 相结合

长期记忆与工作记忆

工作记忆/短期记忆

  1. 工作记忆是负责处理你脑海中即时的有意识的想法的部分，即对新事物是工作记忆在起作用
  2. 工作记忆只能存储大概4个组块的信息，即容量很小
  3. 因为大脑倾向于将记忆归结到组块，所以工作记忆看似比实际容量要大
  4. 可以从长期记忆中提取内容到工作记忆中来
  5. 需要短时间内多次重复才能存储到工作记忆中（例如背电话号码）

所以说短期记忆像是一个 “低效的精神黑板”，用计算机的术语，我更想称之为 “内存”

长期记忆

  1. 长期记忆的容量是巨大的，可以存储数以十亿计的内容
  2. 由于信息之间错综复杂，所以在寻找需要的信息时有一些困难
  3. 只有重复记忆多次才能增加以后需要它想起来的机率
  4. 需要通过时间和练习才能将短期记忆中的内容转换至长期记忆
  5. 短时间内的大量重复记忆不会有什么明显效果，最好是能够 间隔重复（例如使用艾宾浩斯记忆曲线来背单词）

所以说长期记忆更像是一个“记忆仓库”，用计算机的术语，我更想称之为“硬盘”

睡眠

从生物学解释，你为什么要睡觉

简单说，非睡眠时会让大脑产生有毒的物质
大脑如何除掉这些有毒物质呢？
原来当人们在睡觉的时候
你的脑细胞会缩小
这样会增加脑内细胞之间的距离
这就像疏通了一条溪流
液体会在细胞空隙间流过 冲走有毒的物质
所以 睡觉这件事有时候看起来像是浪费时间 却是大脑保持清洁和健康的一种方式
事实上 少睡一点不止会让你在考试中发挥失常
长时间的睡眠不足
更会产生各种不好的副作用 比如头痛
抑郁症 心脏疾病 糖尿病 或 缩短寿命

睡眠对学习的重要性

  1. 你的大脑会将你学习和思考过的想法以及概念进行整理，它会清除掉一些记忆中不太重要的部分，同时增强你需要或想要记住的区域的记忆
  2. 大脑也会排练，你想学习的内容里的困难点，这些点在神经中枢里一遍遍排演以增强加深记忆力
  3. 睡眠对人们区分找出困难问题的能力和理解所学知识的能力有显著影响
  4. 大脑会在睡眠时着重“强化“在睡前学习的内容 似乎从另一个角度说明了 学习中的休息 的重要性

当然，以上的前提是：你首先必须通过做集中精力状态下的工作在脑中搭建发散思维模型

你应该怎么做？

好好学，好好玩

该学的时候好好学习，该玩的时候痛痛快快的玩 果然不是空穴来风

  1. 使用 番茄工作法 来集中精力做好眼前的事情
  2. 阶段性学习结束后（比如完成了一个番茄），为自己空出放松的时间（比如5分钟）
  3. 根据时间表安排来生活，将临时的非重要安排项目后延（这个观点并非出自学习课程，但有助于减少拖延现象）
  4. 对于学过的内容间隔重复以使其成为长期记忆

睡个好觉

更好的睡眠是为了更好的学习

  1. 睡前90分钟尽量少使用电子设备（蓝光会影响睡眠质量）
  2. 最好能够在睡前回顾今天所学的知识，这样能充分利用睡眠带来的记忆福利（纸质书、kindle、自己的笔记本？）
  3. 在困的时候睡觉，以便快速入睡
